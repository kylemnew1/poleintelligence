Layer 1 — Drone Imagery

The Foundational
Data Layer

Every pole record begins here. Drone imagery provides the ground truth layer that validates existence, corrects GIS coordinates, and catalogs every visible component before any other data is collected.

✦ Geo-verification — True up pole coordinates, correcting legacy GIS errors that can be off by meters
✦ Joint-use attachment inventory — Cable sizes, messenger wires, cross arms, transformers, street lights
✦ Down guys & anchors — Guy wire angles, anchor types, support structure components cataloged per pole
✦ Visual inspection — Woodpeckers, splits, leaning, damaged insulators, corroded hardware captured in imagery

Layer 2 — Groundline Inspection

What Drone Eyes
Can't See

The most critical failure zone for any wood pole is underground — where moisture and fungi attack the base. Groundline inspection uses non-invasive technology to generate a complete below-grade health picture.

✦ Full decay profile — Percentage of cross-section lost to rot, mapped at multiple depth intervals
✦ Moisture profile — Moisture content measurement flags early-stage biological degradation before visible damage
✦ Ground-level photography — Anchor confirmations, pole birthmark capture, supplemental images filling drone blind spots
✦ Health indication score — Numeric rating feeds directly into the pole's risk matrix as the structural integrity input
Layer 3 — LiDAR Capture

Three-Mode Point Cloud
Architecture

LiDAR is deployed in a deliberate hierarchy — fastest and broadest first, then gap-filling modes layer in precision where needed. The result: a seamless, complete point cloud ready for measurement and feature extraction.

1
Mobile LiDAR

Vehicle-mounted sensor sweeps entire corridors at highway speeds — the fastest way to build a base-layer point cloud across thousands of poles per day.

🚗  Highest coverage velocity · Best for linear routes
2
Aerial LiDAR

Drone or fixed-wing flyover fills occlusion gaps left by mobile capture — vegetation overhang, complex angles, and span geometry that ground-level sensors miss.

🚁  Gap-filling · Vegetation penetration · Wire geometry
3
Terrestrial LiDAR

Backpack-mounted sensor deployed only where mobile and aerial cannot achieve required density — confined corridors, dense urban canyons, complex equipment arrays.

🎒  Precision fallback · Urban canyons · Complex geometry
📡 Post-collection processing: All three modes are merged, classified (ground / pole / wire / vegetation / hardware), measured, and feature-extracted to produce the geometric inputs required for SPIDAcalc's JSON schema.

Layer 5 — Risk Matrix

The All-in-One Pole Health Index

Every data stream feeds a living risk score — updated in real time as conditions change, weather events occur, or new inspections are completed.

Input 1
Visual Inspection
Drone imagery — damage, lean, hardware condition, attachment anomalies
Input 2
Integrity Score
Groundline decay %, moisture content, health classification (Good / Average / Poor)
Input 3
Load Compliance
SPIDAcalc safety factor result — pass margin, overload %, flagged conditions
Input 4
Vegetation Encroachment
Tree proximity, canopy contact risk, trimming priority scoring from LiDAR + imagery
Input 5
Weather Modeling
Built-in wind / ice / temperature models update pole stress estimates in real time
Input 6
Legacy Data Integration
Asset owner's historical inspections layered in to deepen accuracy and accelerate matrix reliability
🛡️ Output: Every pole receives a composite risk score — Green (healthy), Yellow (monitor), Red (remediate immediately) — giving asset owners a live view of their entire infrastructure's risk exposure across 950,000+ poles.
